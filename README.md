DeteccaoDeFaixas

Detecção automática de faixas de rodagem em vídeo utilizando visão computacional

Visão geral

Este projeto implementa um pipeline completo de visão computacional para detectar automaticamente as faixas esquerda e direita de uma pista a partir de um vídeo gravado em ambiente urbano ou rodoviário.

A solução foi desenvolvida em Python, utilizando as bibliotecas OpenCV e NumPy, com foco em:

processamento de imagens em tempo real

extração de características geométricas

organização modular do código

robustez frente a ruído e variações visuais

O projeto simula, em escala reduzida, um dos problemas fundamentais de sistemas de assistência à condução e veículos autônomos.
